# 🤖 BTC Signal Bot for Polymarket

Бот аналізує BTC на таймфреймах **5m, 15m, 30m, 1h** і надсилає сигнали у Telegram,
коли кілька індикаторів одночасно підтверджують рух.

---

## 📦 Структура файлів

```
btc_signal_bot/
├── main.py           ← запускати цей файл
├── config.py         ← твої налаштування (токен, chat_id тощо)
├── data_fetcher.py   ← завантаження даних з Binance
├── indicators.py     ← RSI, MACD, EMA, Bollinger
├── signal_engine.py  ← логіка якісних сигналів (система балів)
├── telegram_bot.py   ← відправка в Telegram
├── web_app.py        ← веб-дашборд
├── templates/
│   └── dashboard.html
└── requirements.txt
```

---

## 🚀 Крок 1: Встановлення Python

1. Скачай Python 3.11+ з https://python.org/downloads
2. При встановленні обов'язково ✅ **Add Python to PATH**

---

## 🔑 Крок 2: Отримай Telegram токен та Chat ID

### Створення бота:
1. Відкрий Telegram → знайди **@BotFather**
2. Напиши `/newbot`
3. Придумай назву та username (напр. `MyBTCSignalBot`)
4. Скопіюй отриманий **токен** (виглядає як `123456789:ABCdefGHIjklMNOpqrSTUvwxYZ`)

### Отримання свого Chat ID:
1. Напиши своєму боту будь-яке повідомлення
2. Відкрий у браузері: `https://api.telegram.org/botТВІЙ_ТОКЕН/getUpdates`
3. Знайди `"chat":{"id":XXXXXX}` — це твій **chat_id**

---

## ⚙️ Крок 3: Налаштуй config.py

Відкрий файл `config.py` і заміни:

```python
TELEGRAM_TOKEN   = "123456789:ABCdefGHIjklMNOpqrSTUvwxYZ"  # ← свій токен
TELEGRAM_CHAT_ID = "987654321"                               # ← свій chat_id
```

Інші параметри можна залишити за замовчуванням.

---

## 📥 Крок 4: Встанови залежності

Відкрий **командний рядок** (cmd) в папці з ботом і виконай:

```bash
pip install -r requirements.txt
```

---

## ▶️ Крок 5: Запуск на ПК

```bash
python main.py
```

Веб-дашборд буде доступний на: **http://localhost:5000**

---

## ☁️ Крок 6: Безкоштовний хостинг у хмарі (Railway)

### Один раз налаштувати:

1. Зареєструйся на **https://railway.app** (безкоштовно, через GitHub)
2. Завантаж файли на GitHub (або у Railway через drag&drop)
3. Натисни **"New Project"** → **"Deploy from GitHub"**
4. У розділі **Variables** додай:
   - `TELEGRAM_TOKEN` = твій токен
   - `TELEGRAM_CHAT_ID` = твій chat_id
5. Натисни **Deploy**

> Railway дає ~500 годин/місяць безкоштовно — достатньо для постійної роботи.

**Альтернативи:** Render.com, Fly.io — також безкоштовні.

## 🧠 Як працює система сигналів

Бот нараховує бали від 0 до 6 за кожен таймфрейм:

| Індикатор | Умова | Бал |
|-----------|-------|-----|
| EMA Cross | EMA9 перетинає EMA21 | +1 |
| EMA Trend | Ціна вище/нижче EMA50 і EMA200 | +1 |
| RSI Zone | Вихід з перепроданості (<35) або перекупленості (>65) | +1 |
| MACD Cross | MACD перетинає сигнальну лінію | +1 |
| Bollinger | Відбиття від смуги Боллінджера | +1 |
| Volume | Обʼєм > 1.5x від середнього | +1 |

**Сигнал надсилається тільки при 4+ балах** (за замовчуванням).

Змінити поріг можна в `config.py`:
```python
MIN_SCORE = 4  # 3 = більше сигналів, 5 = тільки найкращі
```

## 📊 Що означають сигнали для Polymarket

| Сигнал | Що робити на Polymarket |
|--------|------------------------|
| 🟢 ВГОРУ | Ставити на "BTC буде вище X" до дедлайну |
| 🔴 ВНИЗ | Ставити на "BTC буде нижче X" до дедлайну |

> ⚠️ **Важливо:** Це інформаційний інструмент, не фінансова порада.
> Завжди аналізуй ринок самостійно перед входом у позицію.

## 🔧 Часті питання

**Q: Бот не надсилає повідомлення**
A: Перевір токен і chat_id у config.py. Переконайся, що ти написав своєму боту хоча б одне повідомлення.

**Q: Помилка при pip install**
A: Спробуй `pip3 install -r requirements.txt` або `python -m pip install -r requirements.txt`

**Q: Хочу більше сигналів**
A: Зменши `MIN_SCORE = 3` у config.py

**Q: Хочу тільки найкращі сигнали**
A: Постав `MIN_SCORE = 5` у config.py
